# Sample Fenced and Indented Code

### Fenced Code Formatting

```java 
      class Dummy {
          private final int intValue;

          public Dummy(int intValue) {
              this.intValue = intValue;
          }
      }
~~~

- Item

  ~~~java 
           class Dummy {
               private final int intValue;

               public Dummy(int intValue) {
                   this.intValue = intValue;
               }
           }
  ```
